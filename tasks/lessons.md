# Lessons

## Public Widget Pages Are User-Facing
- Do not include operator-only DM helper copy, share templates, or "DM으로 전달할 링크" wording on widget detail pages.
- If a page is opened by a user from a DM, the page should read as the final user experience: install guidance, script/code area, and clear status only.
- Keep DM automation text in internal docs or tooling, not in public HTML pages.
