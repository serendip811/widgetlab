Scriptable 위젯 설치방법

앱 설치, 스크립트 붙여넣기, 홈화면 위젯 연결까지 순서대로 진행하면 됩니다.

1

앱 준비

Scriptable 설치

  1. App Store에서 Scriptable을 설치합니다.
  2. Scriptable 앱을 한 번 실행합니다.
  3. 오른쪽 위 + 버튼을 눌러 새 스크립트를 만듭니다.
  4. 전달받은 스크립트 코드를 전체 복사해 붙여넣습니다.
  5. 스크립트 이름을 알아보기 쉽게 저장합니다.
2

홈화면

위젯 추가

  1. 아이폰 홈화면의 빈 공간을 길게 누릅니다.
  2. 왼쪽 위 + 버튼을 누릅니다.
  3. Scriptable을 검색합니다.
  4. 중간 크기 위젯을 선택합니다.
  5. 위젯 추가를 누릅니다.
3

연결

스크립트 연결

  1. 홈화면의 Scriptable 위젯을 길게 누릅니다.
  2. 위젯 편집을 누릅니다.
  3. Script 항목을 선택합니다.
  4. 방금 저장한 위젯 스크립트를 선택합니다.
  5. 홈화면으로 돌아와 위젯 표시를 확인합니다.

자주 막히는 지점

위젯이 비어 보이면 여기부터 확인하세요.

  • 스크립트 전체가 빠짐없이 붙여넣어졌는지 확인합니다.
  • 위젯 편집 화면에서 올바른 Script가 선택됐는지 확인합니다.
  • 권한 요청이 뜨면 허용한 뒤 위젯을 한 번 눌러 새로고침합니다.
  • 위젯 코드가 업데이트된 경우 Scriptable 앱에서 기존 코드를 새 코드로 교체합니다.
